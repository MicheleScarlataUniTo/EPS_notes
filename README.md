# Elementi di Probabilità e Statistica – Esercizi in R

La presente cartella contiene una raccolta di esercizi relativi al corso di **Elementi di Probabilità e Statistica**, svolti mediante il linguaggio di programmazione **R**, nell’ambito del corso tenuto presso l’**Università degli Studi di Torino (UniTo)** durante l'anno accademico 2024/2025.

Il materiale è destinato esclusivamente a scopo didattico e di supporto allo studio individuale degli argomenti trattati durante il corso.

⚠️ **Avvertenza**  
Gli esercizi potrebbero contenere errori o imprecisioni.  
Si invita chiunque riscontri problemi o incongruenze a contattarmi per le opportune segnalazioni e correzioni.

Il feedback è gradito e contribuisce al miglioramento del materiale.
